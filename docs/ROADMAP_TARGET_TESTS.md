# Breadth Target Test Roadmap

Objective: run all CVE YAML patterns (`alias=always`) on as many zk0d circuits as possible, sequentially, to observe behavior across diverse targets.

## Execution Profile
- Matrix: `targets/zk0d_matrix_breadth.yaml`
- Registry: `targets/fuzzer_registry.prod.yaml`
- Scheduler: sequential targets (`--jobs 1`), one template process (`--batch-jobs 1`), `--workers 2`
- Breadth budget: `--iterations 1500 --timeout 90`
- Framework coverage: Circom + Noir + Halo2
- Halo2 note: set `RUSTUP_TOOLCHAIN=nightly` when running Halo2 targets against `halo2-scaffold`.

## Step-by-Step Loop (Workers=2)
1. List indexed targets:
```bash
scripts/run_breadth_step.sh --list
```
2. Run one target step:
```bash
scripts/run_breadth_step.sh --step 1
```
3. Write observations in generated file:
- `artifacts/roadmap_step_tests/observations/step_<NNN>__<target>.md`
4. Update the corresponding row remark/check in this checklist.
5. Continue with the next step index.

## Follow-Up Report
- Focused follow-up pass summary: `docs/ROADMAP_TARGET_TESTS_FOLLOWUP.md`

```bash
cargo run --release --bin zk0d_matrix -- \\
  --matrix targets/zk0d_matrix_breadth.yaml \\
  --registry targets/fuzzer_registry.prod.yaml \\
  --jobs 1 --batch-jobs 1 --workers 2 \\
  --iterations 1500 --timeout 90 \\
  --summary-tsv artifacts/zk0d_matrix_external_max_summary.tsv
```
